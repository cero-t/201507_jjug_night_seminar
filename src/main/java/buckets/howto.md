# 問題

3Lのバケツと8Lのバケツがあって、これらを使って4Lの水を得たい。
可能な操作は以下3つ。

- バケツいっぱいに水を満たす
- バケツの水を全て捨てる
- 別のバケツに水を注ぐ ＊1

＊1 : 例えば、(8L / 8L)のバケツから(0L / 3L)のバケツに注ぐと →(5L / 8L)と(3L/ 3L) になります。

本処理の最短解を探索し、結果の出力をしてください。

### 結果出力イメージ

- 3を満たす(3, 0)
- 3から8へ移す(0, 3)
- 3を満たす(3, 3)
- 3から8へ移す(0, 6)
- 3を満たす(3, 6)
- 3から8へ移す(1,8)
- 8を捨てる(1,0)
- 3から8へ移す(0,1)
- 3を満たす(3,1)
